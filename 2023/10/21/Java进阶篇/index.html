Java进阶篇

进程

线程

多线程：独立操作，互不影响，效率较高，利用空闲时间，让程序做多件事情

单线程程序：从头往下，一条条执行，cpu 执行效率较低

并发

同一时刻，多个指令在单个cpu上交替执行

并行

同一时刻，多个指令在多个cpu上同时进行

多线程的实现方式

继承Thread方式

  1. 自定义一个类，实现 Thread
  2. 重写 run方法
  3. start启动线程
